[Myrthel](mailto:m.dogge@uu.nl) and I programmed a function that allows for easy calculations of confidence intervals while accounting for all the factors. The function requires at least one within factor and is a generalized version of [this code](http://www.cookbook-r.com/Graphs/Plotting_means_and_error_bars_(ggplot2)/).

Check out its [repository](https://github.com/Goallab/WithinS_CI) where we have not only the function but also examples.

The function is also included in the R package `goallabr`, which I am in the process of developing. Check out its repository [here](https://github.com/chrisharrisUU/goallabr) if you prefer using a package.
